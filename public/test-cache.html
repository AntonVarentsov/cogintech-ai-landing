Тест кэширования

Эта страница поможет проверить, работает ли кэширование на вашем сервере.

Проверка заголовков кэширования:

Откройте Developer Tools (F12) и перейдите на вкладку Network.

Обновите страницу и проверьте заголовки для следующих файлов:

  • CSS файлы должны иметь: Cache-Control: public, max-age=31536000, immutable
  • JS файлы должны иметь: Cache-Control: public, max-age=31536000, immutable
  • Изображения должны иметь: Cache-Control: public, max-age=31536000, immutable
  • Видео должны иметь: Cache-Control: public, max-age=31536000, immutable

Тестовые файлы:

Проверьте заголовки для этих файлов:

  • CSS файл
  • JS файл
  • Изображение
  • Видео

Инструкции для Timeweb:

Если кэширование не работает:

  1. Убедитесь, что файл .htaccess находится в корневой папке сайта
  2. Проверьте, что в настройках хостинга включена поддержка .htaccess
  3. Обратитесь в техподдержку Timeweb для включения mod_headers
  4. Проверьте логи ошибок сервера
