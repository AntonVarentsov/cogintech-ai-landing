# TaskMaster AI - Руководство по использованию

## 🚀 Обзор
TaskMaster AI настроен для управления миграцией CognitechAI Landing с React на Astro.

**Всего задач:** 11  
**Общее время:** 130 часов  
**Теги:** master  

## 📋 Команды TaskMaster CLI

### Основные команды нашего CLI:
```bash
# Показать статус проекта и статистику
node tm.cjs init
node tm.cjs status

# Список всех задач с приоритетами
node tm.cjs list

# Показать детали конкретной задачи
node tm.cjs show 1                    # По номеру
node tm.cjs show astro-setup-1        # По ID

# Начать работу над задачей
node tm.cjs start 1
node tm.cjs start astro-setup-1

# Завершить задачу
node tm.cjs complete 1
node tm.cjs complete astro-setup-1
```

### MCP TaskMaster команды (если доступны):
```bash
# Добавить задачу из PRD
task-master parse-prd PRD-ASTRO-MIGRATION.md --append

# Добавить задачу с описанием
task-master add-task --prompt "Описание задачи"

# Посмотреть помощь
task-master add-task --help
```

## 🔄 Управление задачами

### Начать новую задачу:
1. `node tm.cjs show <task-id>` - изучить детали
2. `node tm.cjs start <task-id>` - начать выполнение  
3. Работать по подзадачам из markdown файла
4. `node tm.cjs complete <task-id>` - завершить

### Отслеживание времени:
- Каждая задача имеет estimate в часах
- Трекать фактическое время выполнения
- Обновлять estimates по мере необходимости
